# Styling

To change the font size:

```css
.axis text,
.slider text {
  font-size: 18px;
}
```

To change the tick text color:

```css
.axis text {
  fill: red;
}
```

To change the parameter value text color:

```css
.slider text {
  fill: green;
}
```
